Dans ce tutoriel, vous apprendrez comment encoder en MEI une mélodie très simple. Vous découvrirez comment décrire les aspects les plus fondamentaux de la notation musicale sous forme de données structurées afin que les ordinateurs puissent analyser votre code.

Le code que vous écrirez sera immédiatement rendu, et le rendu réagira à vos modifications. De cette manière, vous aurez une idée de la façon dont les différents paramètres interagissent.

Une fois ce tutoriel terminé, vous pourrez avoir envie d'en suivre d'autres qui vous introduiront à d'autres aspects importants de MEI. Ces tutoriels servent également de référence simple pour MEI - vous pouvez toujours y revenir.

Pour commencer, veuillez cliquer sur le bouton « continuer » en bas à droite de ce paragraphe.
