In this tutorial, you will learn how to encode different types of rests from the Common Music Notation repertoire using MEI.

Please consult XML basics and minimal MEI structure and/or the Quickstart tutorial if you need more information to complete the task below.

According to the MEI Specification, a rest is "a non-sounding event found in the source being transcribed". Encoding that is simple: Just use a <rest> element, with a @dur attribute.
