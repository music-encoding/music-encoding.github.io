Please try to reproduce the melody above with MEI. Each note needs @pname, @oct and @dur.

Keep in mind that the @dur attribute can have values like "1", "2", "4", "8" or "16" to indicate different rhythmical values. For the first note, you need an additional @dots attribute, where you need to specify the number of dots as an integer – in this case, it's simply a dots="1".
