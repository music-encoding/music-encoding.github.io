In this tutorial, you will get accustomed with some very basic XML backgrounds and learn about the outermost structure of a valid MEI document. You will learn to identify different parts of an MEI file and which type of information you can expect in which part.

The topics addressed in this tutorial are referring to the chapter Structural Elements of the MEI Guidelines. We recommend to consult this chapter whenever you need more detailed information.

Please be aware that the code you write in this tutorial cannot be immediately rendered because it lacks information about any actual musical content.
