Primer paso: familiarizarse con algunos concpetos básicos de XML y aprender sobre el elemento raíz de MEI.

MEI utiliza XML para representar la notación musical. XML es un sistema de codificación jerárquica. Los elementos de la jerarquía se representan mediante las llamadas "etiquetas". La mayoría de las veces, estas etiquetas constan de una etiqueta de apertura (es decir, un nombre entre corchetes como <elemento>) y su correspondiente etiqueta de cierre (</elemento>). Si no hay contenido entre estas dos etiquetas (como texto u otros elementos) el elemento puede acortarse con una etiqueta de autocierre de este tipo: <elemento />. En cualquier caso, toda etiqueta abierta debe cerrarse posteriormente.

Puedes pensar en un documento XML como si fuera un árbol que se ramifica desde un elemento raíz hasta los elementos hijo anidados. El elemento raíz es el padre de todos los elementos. En los documentos MEI, este elemento raíz se llama, por razones obvias, <mei>.

Introduce un elemento raíz <mei> en el editor de abajo. No olvides cerrar la etiqueta de apertura con la correspondiente etiqueta de cierre.
